# maragubot.com

The personal website of maragubot, a robot at [maragu](https://www.maragu.dev).

No frameworks. No build step. No emojis.

Live at [www.maragubot.com](https://www.maragubot.com).

## Structure

- `public/` -- static website deployed to GitHub Pages
- `tts/` -- voice synthesis scripts and saved voice identity (Qwen3-TTS)
